Anti-theft protection
What it does
Monitors geofence exits and blocks the engine if the object leaves the designated area. All events are forwarded to the tracking endpoint.
How it works
The Logic node checks whether the object has left the geofence. If true, a Device Action blocks the engine. Both branches send data to the Output Endpoint for tracking.
Setup
  1. Open the Data Source node and select the devices or device group you want to monitor.
  2. In the Logic node, set the geofence boundary that triggers the anti-theft action.

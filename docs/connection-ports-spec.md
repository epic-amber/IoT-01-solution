# Спецификация: визуальная обратная связь для соединений нод

## Порты (коннекторы на нодах)

Каждая нода имеет два типа портов:

- **Input** — принимающий порт (слева), обычно с иконкой стрелки/крестика.
- **Output** — исходящий порт (справа): универсальный `+` или пара `then`/`else` для Logic-ноды.

## Три состояния порта

| Состояние                     | Цвет обводки          | Толщина обводки | Цвет иконки внутри      |
| ----------------------------- | --------------------- | --------------- | ----------------------- |
| **Default** (не подключен)    | серый                 | тонкая (1.3)    | серый, штатная толщина  |
| **Active** (во время drag)    | цвет по правилу ниже  | толстая (2)     | тот же цвет, штатная толщина |
| **Connected** (подключен)     | цвет по правилу ниже  | тонкая (1.3)    | тот же цвет, штатная толщина |

Внутренняя иконка меняет только цвет — её толщина линии не меняется никогда.

## Правило цвета

- **Input-порт** → цвет своей ноды (например, purple для Logic, blue для Data Source).
- **Output-порт `then`** → зелёный (семантика success).
- **Output-порт `else`** → красный (семантика fail).
- **Output-порт `+`** → цвет своей ноды.

## Сценарий взаимодействия

1. **Пользователь нажимает на output-порт.**
   Порт-источник переходит в состояние **Active** (свой цвет + толстая обводка). Из него тянется линия за курсором в том же цвете.

2. **Курсор двигается по холсту.**
   Если он попадает на валидный input-порт другой ноды, тот тоже переходит в **Active** (цвет своей ноды + толстая обводка). Линия притягивается к его центру. Все остальные порты не подсвечиваются.

3. **Курсор уходит с валидной цели.**
   Цель возвращается в предыдущее состояние (Default либо Connected).

4. **Пользователь отпускает мышь на валидной цели.**
   Оба порта — source и target — переходят в **Connected** (свой цвет + тонкая обводка). Соединение сохраняется, порт помечается как подключённый.

5. **Пользователь отпускает мышь вне валидной цели.**
   Source-порт возвращается в предыдущее состояние (Default или Connected — если был подключён ранее). Ничего не создаётся.

## Ключевые UX-принципы

- **Никакого glow, тени или свечения.** Только смена цвета и толщины обводки.
- **Никакой массовой подсветки** валидных целей заранее — реагирует только тот порт, над которым курсор.
- **Плавные переходы** цвета и толщины через CSS-transition — движение ощущается «живым», без резких скачков.
- **Семантика цветов** сохраняется: `then` всегда зелёный, `else` всегда красный, во всех состояниях.
